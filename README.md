## Descripción del Proyecto
Este sistema simula la administración de un parqueadero real utilizando una matriz de 16x16. 
Cumple con los requisitos obligatorios:
* *Uso de Funciones:* Lógica modularizada para visualización, ingreso y cobro.
* *Ciclos For:* Generación y recorrido dinámico del mapa.
* *Punteros:* Implementados en la función buscarEspacioLibre para manipulación de memoria.
* *Matriz:* Representación gráfica del parqueadero con colores (Verde: Libre, Rojo: Ocupado).
